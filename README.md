# CasaOferta 

CasaOferta es una pagina web para buscar videojuegos, comparar precios y consultar ofertas en varias tiendas

El proyecto utiliza:

- React
- Node.js
- Express
- MySQL
- Sequelize
- Docker
- RAWG API
- IsThereAnyDeal API


# Caracteristicas

- Busqueda de videojuegos
- Comparador de precios
- Historial de precios
- Alertas de descuentos
- Favoritos
- Juegos destacados
- Categorias
- Cache persistente
- Cron jobs automaticos

# Tecnologias

## Frontend
